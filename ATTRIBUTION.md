# Atribución y procedencia

## Proyecto de referencia

Esta edición toma como punto de partida el repositorio [JoshuaHernandezMartinez/SpaceShipGame](https://github.com/JoshuaHernandezMartinez/SpaceShipGame), específicamente la carpeta `Space Ship Game episode 24_final`.

La versión actual fue adaptada, reorganizada y ampliada para el repositorio [SalazarPaulo/space_ship](https://github.com/SalazarPaulo/space_ship). Las mejoras incluyen interfaz Java2D dibujada por código, escalado de ventana, configuración persistente, estados de pausa y Game Over, puntajes Top 10 con nombre, las mejoras acumulables y ciertas correcciones de estabilidad.

## Recursos gráficos y de audio

- **Space Shooter Redux (plus fonts and sounds)** — Kenney Vleugels / Kenney.  
  Recursos suministrados dentro del proyecto y distribuidos bajo **CC0**. Se conserva el archivo `resources/ATTRIBUTION-KENNEY-CC0.txt`.

La atribución identifica el origen de los recursos y del proyecto de referencia. Antes de reutilizar o redistribuir código procedente de terceros, revisa los términos o licencias correspondientes.
